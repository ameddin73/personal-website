# Personal Website

### Run

`npm start`

### Build

Automagical gitops Netlify don't think about it. Print the resume first tho.

### Print Resume

1. `alias chrome=<path-to-chrome>`
1. `npm start`
1. `npm run print`

#### Print Custom Resume

1. `alias chrome=<path-to-chrome>`
1. `npm run customize -- "goal" '"skill1","skill2"'`
1. `npm start`
1. `npm run print-special`
