Distro-PHL

beta

A classified's platform built for good.

Visit Website

GitHub Repo

Powered by React.

Distro-PHL is a full-featured P2P market designed to connect people with resources free of charge.

Distro-PHL is a free-to-use tool made possible by super-lightweight development.
By writing the frontend in Typescript, logic can be driven safely from the client while the backend is supported entirely by Google's BAAS Firebase.

As a developer this means I can focus on one language, one framework, one API, and one code base.
For users, it means a fast distributed edge node network, reactive content updates, and guaranteed security and data privacy.
